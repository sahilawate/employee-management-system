# Employee Management System

A RESTful **Employee Management System** built using **Java - Spring Boot**, **Spring Data JPA** and **MySQL**.  
The application provides APIs to manage employee records including creating, retrieving, updating and deleting employee information.

## Features
- Employee CRUD operations (Create, Read, Update, Delete)
- RESTful API architecture
- MySQL database integration
- Spring Data JPA for database operations

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Project Overview
This project demonstrates the development of a backend employee management service using the Spring Boot framework. It exposes RESTful APIs that allow clients to perform CRUD operations on employee records stored in a relational database.

## Author
Backend project developed using Spring Boot for learning REST API development.
